## 3. 画面仕様

### 3.1 SCR-001: ログイン画面

**URL:** `/login`  
**メソッド:** GET（表示）/ POST（送信）

#### 表示項目

| 項目 | 種別 | 説明 |
|------|------|------|
| ユーザー名 | テキスト入力 | 必須。最大150文字 |
| パスワード | パスワード入力 | 必須 |
| ログインボタン | ボタン | フォーム送信 |

#### バリデーション

| 項目 | ルール |
|------|--------|
| ユーザー名 | 必須、最大150文字 |
| パスワード | 必須 |

#### 動作

- 認証成功：`next` パラメータのURL または トップページ（`/`）へリダイレクト
- 認証失敗：エラーメッセージ「ユーザー名またはパスワードが正しくありません」を表示し、画面を再表示

---

### 3.2 SCR-002: トップページ

**URL:** `/`  
**要認証:** あり

#### 表示内容

- ナビゲーションバー（各モジュールへのリンク）
- 現在の処理年月
- 各モジュールの最新バッチ件数サマリー

---

### 3.3 SCR-003: 給与データ画面

**URL:** `/salary/`  
**要認証:** あり

#### 構成

1. **アップロードフォーム**（アコーディオン形式）
   - ファイル選択フィールド（.xlsx/.xls のみ）
   - アップロードボタン
   - 既存バッチがある場合：アップロードボタンを無効化し、「既存データを削除してください」と表示

2. **バッチ一覧テーブル**

| 列名 | 内容 |
|------|------|
| ファイル名 | アップロードされたファイル名 |
| 登録日時 | YYYY-MM-DD HH:MM:SS |
| 登録者 | ログインユーザー名 |
| 件数 | 取り込まれた行数 |
| 確認 | 詳細モーダルを開くボタン |
| 削除 | 削除確認モーダルを開くボタン |

3. **給与詳細モーダル**（HTMX: `GET /salary/detail/<batch_id>`）
   - バッチに含まれる全行データの一覧表示

#### 特記事項

- シングルバッチポリシー：バッチが1件でも存在する場合、アップロードフォームを無効にする

## 4. API仕様

### 4.1 認証

#### POST /login

| 項目 | 内容 |
|------|------|
| 概要 | ログイン処理 |
| Content-Type | `application/x-www-form-urlencoded` |
| CSRF | Flask-WTFによるCSRFトークン必須 |

**リクエストパラメータ**

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| username | string | ○ | ユーザー名 |
| password | string | ○ | パスワード |
| csrf_token | string | ○ | CSRFトークン |

**レスポンス**

| 条件 | ステータス | 内容 |
|------|-----------|------|
| 認証成功 | 302 | `next` または `/` へリダイレクト |
| 認証失敗 | 200 | ログイン画面を再表示（エラーメッセージ付き） |

---

#### GET /logout

| 項目 | 内容 |
|------|------|
| 概要 | ログアウト処理 |

**レスポンス:** 302 → `/login` へリダイレクト

---

### 4.2 給与データ

#### GET /salary/

| 項目 | 内容 |
|------|------|
| 概要 | 給与データ画面を表示 |
| 認証 | 必須 |
| レスポンス | 200 HTML |

---

#### POST /salary/upload

| 項目 | 内容 |
|------|------|
| 概要 | 給与データExcelファイルのアップロード |
| Content-Type | `multipart/form-data` |
| 認証 | 必須 |
| CSRF | 必須 |

**リクエスト**

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| file | file | ○ | .xlsx または .xls ファイル |

**レスポンス（HTMX部分更新）**

| 条件 | ステータス | 内容 |
|------|-----------|------|
| アップロード成功 | 200 | アップロード結果HTML（成功件数表示） |
| バリデーションエラー | 200 | エラー詳細HTML（行番号・フィールド・エラー内容） |
| ファイル形式エラー | 400 | エラーメッセージHTML |
| 既存バッチあり | 400 | エラーメッセージHTML |

---

#### GET /salary/detail/`<batch_id>`

| 項目 | 内容 |
|------|------|
| 概要 | 給与バッチの詳細データ取得（モーダル用） |
| 認証 | 必須 |
| レスポンス | 200 HTML（モーダルコンテンツ） |

**パスパラメータ**

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| batch_id | integer | バッチID |

---

#### DELETE /salary/delete/`<batch_id>`

| 項目 | 内容 |
|------|------|
| 概要 | 給与バッチの削除 |
| 認証 | 必須 |
| 補足 | HTMX非対応ブラウザ向けにPOSTメソッドもサポート |

**レスポンス**

| 条件 | ステータス | 内容 |
|------|-----------|------|
| 削除成功 | 200 | 更新されたバッチ一覧HTML |
| 対象なし | 404 | エラーページ |
